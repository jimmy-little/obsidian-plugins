# Ratchet

Suite shell UI for Ratchet (main single view).

## URL schemes (Obsidian URI)

**Plugin id (URI host):** `ratchet`.

Open the main Ratchet leaf:

```text
obsidian://ratchet
obsidian://ratchet?screen=main
```

Conceptual route: `/ratchet/main`.

See **Settings → Ratchet** for the same reference.
